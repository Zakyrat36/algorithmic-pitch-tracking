# Algorithmic Analysis & Pitch Record Tracking System

An advanced data analytics application written in Python designed to ingest dataset records via CSV and benchmark structural algorithm execution times.

## 📊 Core Algorithms Built From Scratch
1. **Sorting Compilers:** 
   * **Bubble Sort:** Iterative neighborhood comparison method executing in $O(n^2)$ time complexity boundaries.
   * **Quick Sort:** High-performance recursive divide-and-conquer processing layout using an array midpoint pivot element.
2. **Search Routines:**
   * **Linear Search:** Sequential index tracking loops to isolate item matches.
   * **Binary Search:** High-efficiency $O(\log n)$ search engine calculating pointer medians (`low`, `mid`, `high`) across structured collections.

## 🛠️ Key Paradigms Used
* **Data Serialization:** Built custom structured data definitions utilizing constructor methods to maps raw string arrays into valid integer, float, and typed object fields.
* **Performance Profiling:** Embedded execution timestamp trackers using Python's `time` utilities to contrast performance differences across complex sorting variations.
